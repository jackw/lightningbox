Lightningbox

Super small, only 6.2kB (2.1kB gzipped).

A small and blazing fast vanilla javascript image viewer.

Star Fork

Why?

It is easy to setup, super small, fast and has no dependencies.

Single Image

// javascript
lightningbox('.gallery-single > a');

// html
<div class="gallery-multiple">
    <a href="/images/1.jpg"><img src="/images/1.jpg" /></a>
</div>

Multiple Images

// javascript
lightningbox('.gallery-multiple > a');

// html
<div class="gallery-multiple">
    <a href="/images/1.jpg"><img src="/images/1.jpg" /></a>
    <a href="/images/2.jpg"><img src="/images/2.jpg" /></a>
